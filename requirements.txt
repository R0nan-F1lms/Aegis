Aegis/
├── data/                  # NEVER EXFILTRATE THIS FOLDER
│   ├── malware/           # Subfolders for classes (e.g., /trojan, /ransomware)
│   └── benign/            # Clean Windows executables
├── src/
│   ├── extractor.py       # Parses PEs and generates CSV
│   ├── classifier.py      # Trains the ML models
│   └── selector.py        # Runs feature selection algorithms
├── output/
│   ├── features.csv       # The safe, exported dataset
│   └── matrices/          # For saving confusion matrix images
└── requirements.txt

[Data Storage]
   ├── /dataset/malware/ (500+ .exe files)
   └── /dataset/benign/  (500+ .exe files)

       ↓
[Phase 1: extractor.py] (Using 'pefile' library)
   ├── Loop through every file in /dataset/
   ├── Try: Parse PE Headers
   │    ├── Extract Imports (DLLs & APIs)
   │    ├── Calculate Entropy of payload
   │    ├── Count Sections & Check Names (.text, .rdata, etc.)
   │    └── Compare VirtualSize vs SizeOfRawData
   ├── Except: Log error (corrupt/unreadable sample)
   └── Output: 'master_feature_matrix.csv'

       ↓
[Phase 2: classifier.py] (Using 'scikit-learn')
   ├── Load 'master_feature_matrix.csv'
   ├── Preprocess: Handle missing values, encode text labels to numbers
   ├── Split Data: 80% Training / 20% Testing
   ├── Train Models: 
   │    ├── Model A (e.g., Random Forest)
   │    ├── Model B (e.g., Support Vector Machine)
   │    └── Model C (e.g., K-Nearest Neighbors)
   └── Output: Print Accuracy, save Confusion Matrices & ROC/PR curves as PNGs.

       ↓
[Phase 3: selector.py] (Using 'scikit-learn' & 'mlxtend')
   ├── Load Data
   ├── Path A: Model-Based Selection (e.g., L1-based feature selection like Lasso)
   │    └── Extract Top 3 feature sets -> Re-train Models -> Output Metrics
   └── Path B: Sequential Feature Selection (SFS)
        └── Forward/Backward selection -> Re-train Models -> Output Metrics
